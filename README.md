# CS Web — Multiplayer FPS

## Запуск

```bash
cd game
npm install
npm start
# Відкрий http://localhost:3000
```

## Структура

```
/game
 ├ index.html   — HTML розмітка (HUD, меню, оверлеї)
 ├ style.css    — Всі стилі
 ├ client.js    — Ігрова логіка (Three.js, Socket.IO клієнт)
 ├ server.js    — Node.js сервер (Express + Socket.IO)
 └ package.json
```

## Мультиплеєр

1. Запусти сервер: `npm start`
2. Гравці відкривають `http://[твій-IP]:3000`
3. В меню → **🌐 МУЛЬТИПЛЕЄР** → вказати нікнейм і кімнату → вибрати команду
4. Гра стартує коли підключаться 2+ гравців

## Боти (одиночна гра)

Просто натисни **ГРАТИ** в головному меню — грає з ботами як раніше.

## Клавіши

- WASD — рух, Shift — спринт, Ctrl — присісти
- ЛКМ — стріляти, ПКМ — приціл (AWP)
- E — плентувати/знешкоджувати
- R — перезарядка, G — кинути зброю
- B — магазин (в фазі купівлі)
- Tab — таблиця результатів (мультиплеєр)
- Y — чат (мультиплеєр)
- 1-5 — вибір слоту зброї

## Модифікація

- Змінити карти/зброю: **client.js** (розділи `MAPS`, `WCAT`)
- Змінити UI: **style.css** або **index.html**
- Змінити серверну логіку: **server.js**
